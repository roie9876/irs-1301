## 1. סקירה כללית

### 1.1 מטרת המוצר
אפליקציה מקומית שמנחה משתמש צעד-אחר-צעד במילוי טופס 1301 (דוח שנתי למס הכנסה).
האפליקציה משתמשת ב-LLM כדי:
- להסביר מה למלא בכל שדה בטופס
- לחלץ נתונים ממסמכים שהמשתמש מעלה (106, 867, אישורים)
- להציע ערכים מומלצים למילוי על בסיס המסמכים
- לענות על שאלות חופשיות בנושא מס

### 1.2 קהל יעד
כל אדם שמגיש דוח שנתי למס הכנסה — שכירים, עצמאים, בעלי הכנסות מעורבות.
הגרסה הראשונה תתמוך בעיקר בשכירים.

### 1.3 עקרונות מנחים
- **הכל רץ מקומית** — אין שרתים בענן, רק קריאות API למודל שפה
- **פרטיות** — מסמכים אישיים נשארים במחשב, לא עולים ל-GitHub
- **תמיכה רב-שנתית** — כל שנת מס = מסמך הנחיות שונה, המערכת מתאימה את עצמה
- **בחירת מודל** — המשתמש בוחר את ספק ה-LLM שלו (OpenAI, Azure OpenAI, Gemini, Claude)
- **עברית** — כל הממשק בעברית, RTL מלא

## 6. טכנולוגיות מוצעות

| רכיב | טכנולוגיה | הערות |
|-------|-----------|-------|
| Backend | Python 3.12 + FastAPI | |
| Frontend | Streamlit | מהיר לפיתוח, תומך RTL |
| LLM Providers | openai, anthropic, google-generativeai SDKs | כל ספק דרך ה-SDK הרשמי |
| OCR/Vision | LLM Vision API (GPT-4o / Gemini) | לקריאת מסמכים סרוקים |
| PDF Processing | PyMuPDF (fitz) + pdfplumber | חילוץ טקסט מ-PDF דיגיטלי |
| RAG Vector DB | ChromaDB | מקומי, אין צורך בשרת |
| Embeddings | OpenAI text-embedding-3-small / local model | |
| Config | python-dotenv + קובץ JSON | שמירת הגדרות משתמש |
